# Clyvernaut M365 Integration Hub

## 📖 Descripción
Repositorio de desarrollo y pruebas para la integración de soluciones corporativas con el ecosistema Microsoft 365. Este proyecto es parte de las iniciativas tecnológicas de **Clyvernaut Inc.** para optimizar flujos de trabajo empresariales mediante automatización y APIs modernas.

## 🚀 Objetivos del Proyecto
- Validar conexiones seguras con Microsoft Graph API.
- Prototipar automatizaciones para gestión de usuarios y grupos en Azure AD.
- Evaluar capacidades de integración con Teams y SharePoint Online.

## 🛠 Stack Tecnológico
- **Lenguaje:** Node.js / JavaScript
- **APIs:** Microsoft Graph API v1.0
- **Autenticación:** OAuth 2.0 (Client Credentials / Delegated)
- **Entorno:** Microsoft 365 Developer Program (E5 Sandbox)

## 📂 Estructura de Pruebas
- `test.js`: Script de validación de conexión y lectura de perfil de usuario.
- `config.sample.js`: Plantilla de configuración (sin credenciales reales).
